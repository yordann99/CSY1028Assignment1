WEB DEVELOPMENT GLOSSARY OF KEY TERMS

Below is a list of terms and their definitions which are all used in web design and development:

CSS Terms:

1. PROPERTIES

Properties are defined within selectors by defining a property and a value. They are separated with a colon and delineated with a semi-colon.

2. PADDING

The padding is the spacing between the content and the border (edge of the element.). We can adjust this value with CSS to move the border closer to or farther from the content.

3. MARGIN

The margin is the space around the element. The larger the margin, the more space between our element and the elements around it. We can adjust the margin to move our HTML elements closer to or farther from each other.

4. FONT FAMILY

The font-family property sets the font of an HTML element's text.

5. SELECTORS

Selectors are used in CSS to select the parts of the HTML that are being styled. You can use several different methods for selecting an element.

6. CLASS NAME SELECTORS

Class selectors select all elements with a matching class.

7. ELEMENT SELECTORS

These can be selected by choosing HTML elements first by simply using the name of the element.

8. ID SELECTORS

ID selectors are used to select only a single item on a page. ID selectors will only select the first element with a matching ID.

9. ATTRIBUTE COLLECTORS

HTML elements are also able to be selected by their attributes.

10. UNIVERSAL SELECTOR

The universal selector may be used to select all the elements in a particular range.

11. PSEUDO CLASS SELECTORS

Pseudo Selectors can be used to narrow down a selection with certain rules.

HTML Terms:

1. CLASS

HTML elements can have one or more classes, separated by spaces. You can style elements using CSS by selecting them with their classes.

2. ID

A HTML element can have an ID attribute to identify it. ID elements should always be unique to that single element, and each element should never have more than one ID.

3. HREF

Links tell the browser where to go using an href attribute, which stores a URL.

4. BASIC FORMATTING

You can easily format text to be bold, italic, or underlined using simple formatting tags.

5. BODY

The body is the container for all of a page's content. Comes after the head tag, within the overall html tag.

6. CHILDREN

An element that is an immediate descendent of another element or nested within another element is called a child. These become useful when using CSS child selectors and psuedo-elements.

7. COMMENTS

HTML comments are sometimes used in code to explain parts of the markup. They are similar to comments in other languages. Users do not see comments in their browser.

8. DIV

A block level container (or 'division' of the web page) for content with no semantic meaning.

9. HEAD

Tag that surrounds important content that is invisible to the user, but is important to the browser.

10. HEADINGS

Heading elements like h1, h2, h3 etc. allow you to use six levels of document headings, ranging from largest to smallest, breaking up the document into logical sections.

11. HORIZONTAL RULES

This tag creates a black line one pixel thick that runs the all the way across its container. It can be styled to look differently with CSS.

12. HTML

HTML stands for Hyper Text Markup Language. It is the language used to create all websites.

13. HTML TAG

All HTML files live within an over-arching html tag. This is the basic tag that defines an html document.

14. HYPERLINKS

Hyperlinks (or just links) take the user to another webpage when they click on it. The most common attribute used with links is href, which tells the browser where the link goes.

15. IMAGES

The img tag embeds an image into your HTML. Always found with the 'src' attribute, which tells the browser where to find the image.

16. LINE BREAKS

This tag is used in a block of text to force a line break. This is to be used for things which are a single paragraph, but where this formatting is necessary such as poems or addresses. To separate paragraphs, separate each paragraph into a separate element instead.

17. LINKS

Link elements are used to connect your document to a related resource (very different from hyperlinks, which take you to another webpage when you click on them). Links appear only in the head section of a document so they do not alter the content, but only the presentation. Links are most commonly used to connect to a stylesheet, script, favicon, or alternate format of the page such as an RSS feed or PDF.

18. LISTS

HTML supports two kinds of lists: ordered lists and unordered lists. Within lists each individual list item has its own tag.

19. UNORDERED LIST

Unordered lists are just lists whose items are denoted with bullet points.

20. ORDERED LIST

Ordered lists' items are denoted with numbers.

21. PARAGRAPHS

One of the most common tags in HTML(p and /p) - it denotes a paragraph of text.

22. SEMANTIC FORMATTING

The difference from formatting tags is that these tags have semantic value. em is used for something that you wish to emphasize and strong is used for something that is important. With both of these elements, you can convey the level of emphasis or importance with nesting.

23. TABLES

An element for displaying information in rows and columns. Supports headers and footers for labeling columns.

24. TAGS AND ELEMENTS

Tags are basic labels that define and separate parts of your markup into elements. They are comprised of a keyword surrounded by angle brackets. Content goes between two tags and the closing one is prefixed with a slash.

25. TITLE

This tag tells the browser what to display as the page title at the top and tells search engines what the title of your site is.
