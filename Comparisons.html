WEB DEVELOPMENT JOB COMPARISONS

WEB DEVELOPER / FRONT END / UI DEVELOPER, (JAVASCRIPT, HTML, CSS)in Nottingham

For this job, the applicant will need previous expierience in UI work and also in different web development sections such as HTML5 and CSS coding. Applicants with UI design experience in JavaScript will also be consider as this is also a good skill to have. The applicant will have to be able to work in a fast paced working environment as well as being able to manage multiple subjects at the same time. The starting salary for an employee at this job is between £25,000 and £30,000.

JUNIOR WEB DEVELOPER in Dundee

For this job, the ideal candidate will have previous experience in working with Javascript and have the ability to work with open source frameworks. An applicant is also required to have good verbal and written communication skills. Main duties of this job role are designing and developing company websites and providing valid maintenance to manage it. This junior web development job will have a starting salary of £20,000 to £25,000 per year.

FRONT END WEB DEVELOPER in Cambridge

For this job, applicants need to have previous experience with Typescript and Javascript. Any other web development experience is beneficial. This company is looking for an individual with strong skils in developing user interfaces and user interface solutions. In this job you will be required to create products that are used in healthcare education and finance. For this job, an employee will usually receive around £25,000 to £34,000 a year with other benefits.
