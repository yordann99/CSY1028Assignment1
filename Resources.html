Useful Resources

Below are some of the most useful resources that can be used when learning to code:

Code Academy



This is a website that you can use to learn several different skills to do with HTML and CSS. You can start a course from the beginning and learn your way throughout the entire course at your own pace. Some courses included in the website are 'Learn HTML', 'Learn CSS', 'Make a website' and 'Deploy a Website' which will all take you through the stages of learning code and using it to develop your own website as well as publishing them.

W3Schools


W3Schools is a website that can be used to watch tutorials and practice coding that you may need help with. It also teaches you everything you need to know about HTML and even has help for when creating your own website.

CodePen


CodePen is a website which allows you to create web pages and other projects using HTML, CSS and JavaScript. This is a goood website to use to test your code as well as use other people's work for inspiration for your own projects.

Code Camp


Code Camp is a useful website that allows you to learn to code for free! As well as this, it has a section on the website that allows you to practice what you have learnt by allowing you to code something using HTML and CSS for yourself.

Code Conquest


Code Conquest is a cdoing website that allows you to learn the basics of coding. This website is mainly aimed at beginners in coding(such as myself) but it can be a very useful way to gain some knowledge about where to start with code if you are completely stuck.

GitHub


GitHub is mainly used for publishing websites but it can be used as a way of learning about code as well. Websites found on there can be used as inspirations for what you may want to achieve with your own website. The positive about this is that you can view the code for any site that is published on there allowing you to be able to get ideas for your own website as well as possibly learn something you were not aware of before.
